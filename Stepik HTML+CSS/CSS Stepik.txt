!!!Составные селекторы!!!

!Группировка селекторов!

Один и тот же набор свойств можно применить к разным селекторам.

button,
.button,
.cta-button{}

Нужно указать через запятую все селекторы, к которым ты хочешь
применить стили.

!Элемент с классом!

Можно стилизовать конкретный элемент, если у него есть определённый
класс:
p.example {}

Селектор выберет все p, у которых есть класс example
.main.active {}

Селектор выберет все элементы с классом main, у которых также
есть и класс active. Пример:
<div class="main active">Пример</div>
HTML:
<div class="main active">Пример 1</div>
<div class="active">Пример 2</div>
CSS:
.main.active{
	color: red;
}
.active{
	color: blue;
}
Слово Пример 1 будет красное, Пример 2 будет синее.

!!!Вложенность!!!

Эти селекторы используются для выбора элементов, которые
вложены в другие элементы.

!Вложенность на любом уровне!

Если между двумя простыми селекторами стоит пробел и больше
ничего, это означает вложенность на любом уровне:
.page p {
	text-decoration: underline;
}
Этот селектор применить подчёркивание ко всем <p> внутри
элемента с классом page на любом ур-не вложенности.

<div class="r">
	<p class="t">Как писал Пелевин:</p>
	<div>
		<p class="r1">
			Человек привык видеть дьявола
			везде, кроме зеркала и телевизора.
		</p>
		<p class="r2">
			Человек привык видеть дьявола
			везде, кроме зеркала и телевизора.
		</p>
	</div>
<div>

HTML:
<p class="tag">Привет!</p

.tag {
	color: red;

!!! Как понимать составные селекторы!!!

Чтобы понимтаь их смысл, нужно читать их справа-налево:

.page > .part {}

Этот селектор выберет все элементы .part, оторые находятся
на первом уровне вложенности в .page

Здесь и дальше, вместо того, чтобы писать элемент с
class="part", буду писать просто .part - точка вначале -
имя класса.

.main .side-menu .menu-item{}
Селектор применится ко всем элементам .menu-item, которые
находятся внутри элементов .side-menu, которые находятся
внутри элементов .main

!!!Это не часто используемые селекторы, можно использовать
материал, как справочник!!!

!Элменты с определенным атрибутом!
Можно задавать стили в зависимости от конкретного
значения конкретного атрибута:
input[type="password"] {}
Этот селектор выберет все элементы <input type="password">

!Первый соседний элемент!

Для того, чтобы выбрать элемент, который находится сразу 
после другого элемента, используется знак +.
Пример:

img + p {
	margin-top: 0;
	font-style: italic;
}
Этот код применит стили во всех <p>, которые идут СРАЗУ после
<img>:

<img
src="https://www.google.ru/images/branding/googlelogo/1x/googlelogo_color_272x92dp.png" />
<p>Это лого гугла</p>
<p>А это просто абзац</p>

!Любой соседний элемент!

Для того, чтобы выбрать элемент, который находится после
другого элемента, используется ~:
img ~ p {
	margin-top: 0;
	font-style: italic;
}
Этот код применит стили во всех <p> после <img>, даже если
между img и p есть другие элементы.
